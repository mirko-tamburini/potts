
## Monte Carlo numerical simulation of the 3-state Potts model in D=2 dimensions

This repository provides the source code for a numerical simulation of the 3-state Potts model, performing computations based on the Markov Chain Monte Carlo algorithms. 

The system configuration consists of a 2D square lattice with spins which take 3 possible values (i.e., 0, 1, and 2) and are updated using a local Metropolis algorithm.
